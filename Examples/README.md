# Examples

Real-world HTML-to-Markdown conversion examples demonstrating the library's features.

| Example | Demonstrates |
|---|---|
| [01-basic-conversion](./01-basic-conversion/) | Core API: headings, lists, links, code blocks, blockquotes |
| [02-vita-with-frontmatter](./02-vita-with-frontmatter/) | `FrontmatterPlugin` + exclude nav/footer |
| [03-wikipedia-article](./03-wikipedia-article/) | Tables, domain-relative links, exclude selectors |
| [04-exclude-navigation](./04-exclude-navigation/) | Blog post with nav/sidebar/footer excluded |
| [05-custom-plugin](./05-custom-plugin/) | Writing and registering a custom plugin |

## Running the examples

Each example's `code.swift` shows the exact Swift code needed. To run:

```swift
import HTMLToMarkdown

// See code.swift in each subdirectory for the full example
```
